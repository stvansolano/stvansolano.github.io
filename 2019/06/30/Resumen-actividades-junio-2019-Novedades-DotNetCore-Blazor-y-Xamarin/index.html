Resumen - Mis actividades junio 2019 - DotNetCore, Blazor y Xamarin

Comparto las diapositivas, links, codigo en GitHub y detalles de las actividades en que participe en junio 2016 sobre .Net Core, Blazor y Xamarin con la comunidad .NET / Mobile Costa Rica. Gracias a todos por acompañarnos!

Estaré actualizando este post con todo el material en los próximos días (código y links de referencia)

Intro a PWAs con DotNet y WebAssembly (TechnoCrux 2019)

Por segunda ocasión fui invitado por la empresa Crux Consultores para dar una charla sobre tecnología .NET y web en el hotel Marriot en Belén, Costa Rica. Aproveché la ocasión para presentarles Blazor, el nuevo framework de Microsoft para la creación de aplicaciones web con HTML, CSS y C#, el cual permite hacer uso de tecnologías web modernas como Bootstrap, WebAssembly y .NET Core desde un mismo lugar lenguaje sin necesidad de (mucho) JavaScript.

Como parte de la actividad tuve demos y presentación de los principales conceptos de Blazor aplicados A Progressive Web Apps (PWA), la nueva forma de crear aplicaciones para desktop y mobile.

C# de Punta a punta con .NET Core en el CPIC

Por segundo mes consecutivo, tuvimos otra sesión de Mobile CR Developers en el CPIC para hablar de C# de principio a fin, desde la creación de backends con .NET Core y conectividad con Xamarin desde el mismo equipo mediante la creación de una API REST, así como herramientas y prácticas más modernas como el uso de contenedores Docker con Azure con ACI (Azure Container Instances).

Tuvimos una gran participación en el Colegio, a quienes les agradezco muchísimo por el apoyo y la invitación a realizar. Espero poder repetir la sesión de manera remota, a petición de la comunidad ya que no tuvimos la oportunidad de hacer streaming y varios en el grupo de Meetup no pudieron participar.
