La espera se acaba. ¡Llega Visual Studio 2019! Eventos de lanzamiento global

Ya está disponible Visual Studio 2019. Participa en los eventos de lanzamiento en tu comunidad y a nivel mundial! Guarda la fecha.

El 2 de abril de 2019 será el lanzamiento oficial de Visual Studio 2019 con demos y conversaciones sobre todo lo nuevo en una de las harramienta más famosas de desarrollo de Microsoft para diferentes plataformas, lenguajes y frameworks de desarrollo de software.

Acceso al evento en linea:

https://visualstudio.microsoft.com/vs2019-launch/

Participa en las meetups cercanas a tu ciudad

Durante este mes y hasta junio de 2019 habrán eventos y conversaciones sobre todo lo nuevo en desarrollo con Visual Studio 2019, Azure DevOps y GitHub tanto en línea como presencialmente.

Tanto el lanzamiento como el IDE están rientados a desarrolladores de diferentes tecnologías web, mobile y cloud en diferentes lenguajes y tecnologías: JavaScript, C++, Python, y .NET C#/Visual Basic.

Acompañanos en la sesión ya sea ponerse al día, ver o probar la última versión de Visual Studio 2019, responder preguntas frequentes y tips.

Eventos locales de lanzamiento de Visual Studio

Puedes ver la lista completa en el siguiente link:

https://visualstudio.microsoft.com/vs2019-launch/local-events/
