Grabacion TechJam Nicaragua 2020 - Mobile Backends Docker, .NET Core, Xamarin y SQL Server en Linux

Comparto la grabación de mi charla del sábado 22 de febrero de 2020, donde la oportunidad de asistir como ponente remotamente en TechJam Nicaragua 2020.

Sobre el evento

TechJam 2020 fue evento que consistió en 3 hilos de conferencias como parte de un mismo evento, interactuando con expositores de Nicaragua e invitados especiales incluyendo Microsoft MVPs y miembros de la .NET Foundation.

Mi charla se tituló: Mobile Backends - Docker, .NET Core + SQL Server (Linux)

Como parte de la charla, mostré la utilización de Docker para crear imágenes de Docker basadas en SQL Server sobre linux, crear un backend de .NET Core y conectarlo con una aplicación Xamarin.Forms
