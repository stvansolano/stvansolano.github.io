El próximo 17 y 27 de febrero 2016 haremos una serie de charlas sobre desarrollo ASP.NET Core 1.0 y Xamarin en las oficinas de Microsoft Costa Rica en Escazú y el Colegio Universitario de Cartago. ¡Invitados-as!

Sobre ASP.NET Core 1.0:

ASP.NET Core 1.0 (anteriormente ASP.NET 5 o vNext) incorpora nuevos cambios y mejoras para el desarrollo web y la posibilidad de desarrollar aplicaciones ASP.NET en Linux y OSX, así como la integración a las herramientas más comunes hoy en día para desarrollo web incluyendo NodeJS, Bower y Gulp.

Conoce más sobre Yeoman, la famosa herramienta de generación de código para NodeJS y aprende a combinarla con Gulp, Bower para crear aplicaciones web con HTML, CSS y JavaScript de forma rápida y sencilla sin salir de tu editor de texto favorito.

MVC 6 con yeoman/bower/gulp Organiza: CR Developers .Net Fecha: Miércoles 17 de febrero de 2016 Lugar: Escazú, Plaza Roble edificio el Patio, piso 2, San José Registro: Meetup

Xamarin + ASP.NET Core: Servicios REST con Web API Organiza: Xamarin Mobile .Net Dev Group Fecha: Sábado 27 de febrero de 2016 Lugar: Sala de Reuniones, Biblioteca del Colegio Universitario de Cartago Registro: Meetup

Tendremos demos, código abierto y rifas entre todos los asistentes a ambas charlas. Los ganadores serán anunciados el sábado 27 de febrero.

Te esperamos!
